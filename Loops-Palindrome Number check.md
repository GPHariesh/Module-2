## Loops in Python: Palindrome Number Checker

## 🎯 Aim
To write a Python program that checks whether a given number is a **palindrome** using loops.

## 🧠 Algorithm
1. Get input from the user and assign it to a variable `a`.
2. Convert the input to palindrome using `::-1` and assign it  to variable b.
3. Check whether a is equal to b
4. If a is eqaul to b print `The given number {a} is a Palindrome`
5. If a is not equal to b print  `The given number {a} is a Palindrome`
